<h1>osTicket - Ticket Lifecycle</h1>

This project demonstrates the lifecycle of a support ticket in <b>osTicket</b>, from ticket creation through assignment, troubleshooting, and resolution.

<br />

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines)
- Windows 11
- Remote Desktop (RDP)
- osTicket

<h2>Operating System Used</h2>

- Windows 11 Pro

<h2>Ticket Lifecycle Stages</h2>

- Ticket Creation
- Ticket Assignment
- Troubleshooting
- Ticket Resolution
